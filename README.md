My personal blog/website, powered by [Gatsby](https://www.gatsbyjs.org)

## Quick start

```bash
npm install && npm run develop
```

then open `https://localhost:8000` or `https://localhost:8000/__graphql` for graphql query interface

## What's using?

-   React
-   Gatsby
-   GraphQL
-   Prism.js
-   Disqus
